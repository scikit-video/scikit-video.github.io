skvideo.datasets.fullreferencepair¶

skvideo.datasets.fullreferencepair()¶

Test video data for full-reference quality algorithms

The sequence “carphone” is provided for quality-based test cases where two videos of the same dimensions are needed.

Returns:

paths : ndarray, shape (2,)

First element contains the absolute path to a pristine video. Second element contains absolute path to a distorted video.
