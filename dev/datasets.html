Test Datasets¶

skvideo.datasets provides several test videos for benchmarking, testing, and example code.

Big Buck Bunny sequence¶

The full absolute path to bigbuckbunny comes bundled with the skvideo install and can be found using skvideo.datasets.bigbuckbunny

Carphone pristine and distorted sequences¶

The path to the carphone pristine and distorted sequences come bundled with the skvideo install and can be found using skvideo.datasets.fullreferencepair
